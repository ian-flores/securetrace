# Add or Update Model Pricing

Register custom model pricing (per 1M tokens).

## Usage

``` r
add_model_cost(model, input_price, output_price)
```

## Arguments

- model:

  Model name string.

- input_price:

  Price per 1M input tokens in USD.

- output_price:

  Price per 1M output tokens in USD.

## Value

Invisible `NULL`.
